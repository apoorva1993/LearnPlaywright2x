# VS Code Keyboard Shortcuts for Mac

## General
- `Cmd + Shift + P` - Command Palette
- `Cmd + P` - Quick Open
- `Cmd + Shift + N` - New Window
- `Cmd + W` - Close Window
- `Cmd + ,` - Settings

## Editing
- `Cmd + Z` - Undo
- `Cmd + Shift + Z` - Redo
- `Cmd + X` - Cut
- `Cmd + C` - Copy
- `Cmd + V` - Paste
- `Cmd + Shift + K` - Delete Line
- `Cmd + Enter` - Insert Line Below
- `Cmd + Shift + Enter` - Insert Line Above
- `Cmd + /` - Toggle Line Comment
- `Cmd + Shift + /` - Toggle Block Comment
- `Cmd + A` - Select All
- `Cmd + D` - Add Selection to Next Find Match
- `Cmd + K Cmd + D` - Move Last Selection to Next Find Match

## Navigation
- `Cmd + G` - Go to Line
- `Cmd + Shift + O` - Go to Symbol
- `Cmd + T` - Go to File (by name)
- `Cmd + Shift + M` - Show Problems
- `Cmd + Shift + J` - Toggle Panel
- `Cmd + J` - Toggle Integrated Terminal
- `Cmd + B` - Toggle Sidebar

## Find and Replace
- `Cmd + F` - Find
- `Cmd + H` - Replace
- `Cmd + Shift + F` - Find in Files
- `Cmd + Shift + H` - Replace in Files

## Multi-cursor and Selection
- `Cmd + Shift + L` - Select all occurrences of current selection
- `Cmd + F2` - Select all occurrences of word at cursor
- `Cmd + Option + Up` - Add cursor above
- `Cmd + Option + Down` - Add cursor below
